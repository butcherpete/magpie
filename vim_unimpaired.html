Vim Unimpaired¶

This plugin provides several pairs of bracket maps. Edit me

tpope/vim-unimpaired

The following maps all correspond to Normal mode commands. If a count is given, it becomes an argument to the command. A mnemonic for the “a” commands is “args” and for the “q” commands is “quickfix”.

Key Action  
[a :previous  
]a :next  
[A :first  
]A :last  
[b :bprevious  
]b :bnext  
[B :bfirst  
]B :blast  
[l :lprevious  
]l :lnext  
[L :lfirst  
]L :llast  
[<C-L> :lpfile  
]<C-L> :lnfile  
[q :cprevious  
]q :cnext  
[Q :cfirst  
]Q :clast  
[<C-Q> :cpfile (Note that only works in a Terminal if you disable flow control.)
]<C-Q> :cnfile  
[t :tprevious  
]t :tnext  
[T :tfirst  
]T :tlast
