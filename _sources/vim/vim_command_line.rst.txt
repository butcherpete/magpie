Vim Command Line
================

A crib of Vim tips and tricks. 

Exiting the shell returns to vim.

To run Linux commands within Vim using the bang commmand:

.. code-block:: rest

   :! command

To start a specific shell within Vim use the band command:

.. code-block:: rest

   :! bash

Or:

.. code-block:: rest

   :sh[ell]
