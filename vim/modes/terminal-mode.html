http://vimcasts.org/episodes/neovim-terminal/

Launch the Shell¶

You can supply a command to the Terminal emulator.

:terminal
:te
:terminal top
:terminal rails server

To open nvim in Terminal mode:

$ nvim +terminal

Activate Terminal Mode¶

To activate terminal mode, press i, a, I, or A.

In Normal mode, these commands switch to Insert mode.

Switch Back to Normal Mode¶

<C-\><C-n>
<Esc>

Set mapping to escape Terminal mode.

Open Terminal Buffers in Windows¶

Run the commands:

:split
:te

The table summarizes commands:

Command Effect
:terminal {cmd} Create terminal buffer in current window
:split | terminal {cmd} Create terminal buffer in horizontal split
:vsplit | terminal {cmd} Create terminal buffer in vertical split
:tabedit | terminal {cmd} Create terminal buffer in new tab page
